## Git и VERSION

- Приоритет: **`Шаблон_Сохранения.md`** + `.cursor/rules/save-on-code-change.mdc`
- После порции правок: **VERSION + git commit** в той же сессии — **не ждать** «закоммить»
- Исключение: **«без коммита»** в текущем чате
- **Push** — только по явной просьбе
- Формат коммита: `vX.Y.Z: тип - кратко` + 1–3 пункта; кириллица через `git commit -F file.txt` (UTF-8)

## Правки кода

- **Не трогать файлы** без явной задачи на изменение
- Диагностика / объяснение — **без patch**, пока не скажешь «исправь», «сделай»
- **Минимальный дифф** — только то, что нужно для задачи
