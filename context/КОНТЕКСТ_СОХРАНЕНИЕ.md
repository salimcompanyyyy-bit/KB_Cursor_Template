# Сохранение — KB_Шаблон

Указатель. Полные правила: **[`Шаблон_Сохранения.md`](../Шаблон_Сохранения.md)**  
Cursor rule: [`.cursor/rules/save-on-code-change.mdc`](../.cursor/rules/save-on-code-change.mdc)

## Принцип

**Изменено → проверено → закоммичено.**

1. `VERSION` — +1 PATCH  
2. `КОНТЕКСТ_АГЕНТА.md` — если менялось поведение  
3. `git commit` по формату из шаблона  
4. В ответе: **vX.Y.Z + хеш**

Исключение: «без коммита» в текущем чате. Push — только по просьбе.

## Файлы context/

| Файл | Зачем |
|------|--------|
| `КОНТЕКСТ_АГЕНТА.md` | Что сделано, как устроен **этот** проект |
| `КОНТЕКСТ_ПОЛЬЗОВАТЕЛЯ.md` | Твои prefs |
| `КОНТЕКСТ_ИДЕЙ.md` | Бэклог (только нереализованное) |
| `АРХИВ/` | Закрытые спеки |

**Не копировать** context из других проектов (MIB и т.д.) — писать под KB_Шаблон.
