# KB_Шаблон

Готовый шаблон для новых проектов с настройками Cursor-агента (как в KB_MIB_PARSER).

## Использование

1. **Скопируйте всю папку** `KB_Шаблон` → переименуйте под проект.
2. Откройте в Cursor → следуйте [`context/ПРОМПТЫ_ПЕРЕНОС_В_НОВЫЙ_ПРОЕКТ.md`](context/ПРОМПТЫ_ПЕРЕНОС_В_НОВЫЙ_ПРОЕКТ.md).

## Структура

- `.cursor/rules/` — правила агента
- `Шаблон_Сохранения.md` — VERSION, коммиты
- `context/` — память, планы, промпты
